h2. modipyd

How this software works:

# Generating python module dependency graphs
# Monitoring module (file) modification
# Notifying modified and dependent modules

So I named it **modipyd** _(modified + python)_.

This project includes **autotest** which is a useful tool for Test-Driven Development.
The **autotest** program continually runs tests based on files you've changed.


h3. Requirements

* Python 2.4 or higher


h3. Developer Notes

* @make lint@ runs "pylint":http://www.logilab.org/857 with configuration file @.pylintrc@ in top level directory
